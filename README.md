# Student Management API

A simple FastAPI project using SQLAlchemy and MySQL to perform CRUD operations on student records.

---

## 🚀 Features

- Create Student
- Get All Students
- Get Student By ID
- Update Student
- Delete Student

---

## 🛠️ Tech Stack

- FastAPI
- SQLAlchemy
- MySQL
- Pydantic
- Uvicorn
